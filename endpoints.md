# Endpoints do Email Web Service

Nós temos que completar essas tarefas:

- [ ] Enviar mensagem
- [ ] Listar mensagens
- [ ] Apagar mensagens
- [ ] Abrir mensagem
- [ ] Encaminhar mensagem
- [ ] Responder Mensagem

## Enviar mensagem

- Será um POST
- Endpoint: /messages

## Listar mensagens

- Será um GET
- Endpoint: /messages

## Apagar mensagens

- Será um DELETE
- Endpoint: /messages/:id

## Abrir mensagem

- Será um GET
- Endpoint: /messages/:id

## Encaminhar mensagem

- Será um PATCH
- Endpoint: /messages/:id

## Responder mensagem

- Será um PUT
- Endpoint: /messages/:id

## Login

- Será um POST
- Endpoint: /login
